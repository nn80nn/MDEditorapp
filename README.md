# MDEditor — Android приложение

Markdown редактор для Android. Пишешь текст в md-формате, смотришь превью, можно вставлять формулы LaTeX, фото и графики.

## Что умеет

- редактор с подсветкой и превью
- вставка формул через MathJax (рендерится в превью)
- вставка фото из галереи
- построение графиков (столбчатый и линейный)
- сохранение на сервер и скачивание обратно
- автосохранение черновика каждые 10 сек

## Стек

- Kotlin + Jetpack Compose
- Retrofit для запросов к серверу
- WebView + markdown-it для превью
- MPAndroidChart для графиков
- JWT авторизация

## Сборка

Нужен Android Studio. minSdk = 28.

```
./gradlew assembleDebug
```
